## Getting Started

Setting up sicun zhang is straightforward. Follow these steps to get started quickly.

<Steps>
  <Step title="Installation" icon="download">
    Install the package using npm or pip:
    <CodeGroup tabs="npm,pip">
      ```bash
      npm install sicun-zhang
      ```
      ```bash
      pip install sicun-zhang
      ```
    </CodeGroup>
  </Step>
  <Step title="Configuration" icon="settings">
    Configure your environment settings:
    ```bash
    export SICUN_ZHANG_ENV=production
    ```
  </Step>
  <Step title="Launch" icon="rocket">
    Start the application with the launch command:
    ```bash
    sicun-zhang start
    ```
  </Step>
</Steps>
